# Darwin Platform Assets

macOS deployment assets for `socks2proxy`.

Included:

- `launchd/org.socks2proxy.plist`: launchd service template
- `scripts/install-launchd.sh`: install/load launchd helper
- `scripts/uninstall-launchd.sh`: unload/remove launchd helper

These files are templates and should be reviewed before production use.
