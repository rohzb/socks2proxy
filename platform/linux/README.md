# Linux Platform Assets

Linux deployment assets for `socks2proxy`.

Included:

- `systemd/socks2proxy.service`: standard service unit
- `systemd/socks2proxy.hardened.service`: hardened service unit
- `apparmor/usr.local.bin.socks2proxy`: AppArmor profile
- `scripts/install-hardening.sh`: hardened install helper
- `scripts/uninstall-hardening.sh`: hardened uninstall helper
- `HARDENING.md`: hardening guide and rationale
